Just follow few steps to download this extension ❤️

step 1

Download the raw folder from github by clicking this link.

step 2

Go to "chrome://extensions/" on your chrome browser.

step 3

Click on "Load unpacked" on the left top of the page and select the "build/chrome-mv3-prod" folder downloaded in the first step.

step 4

Your extension is ready to launch so pin it for the easy access.

step 5

For an api key visit this link and create one new secret key and paste it over there.

step 6

Now you are ready to use
